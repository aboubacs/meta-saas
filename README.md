## Meta-SaaS

A set of common features for any SaaS application, out of the box, with API integrations.

### Features 

- User management
  - Authentication
    - Signup/Signin
      - Email/Password
      - oauth (Google, Facebook, Twitter, Linkedin, Github...)
    - 2FA
    - Account recovery
  - Authorization
    - Roles
    - Permissions
    - Groups
- Subscription
  - Billing
  - Plans
  - Invoicing
- Analytics
  - Usage
  - KPIS
  - Dashboards
- Notifications
  - Email
  - SMS
