## Meta-SaaS

A set of common features for any SaaS application, out of the box, with API integrations.

### Main Features

- User management
  - Authentication
    - Signup/Signin
      - Email/Password
      - Social logins (Google, Facebook, Twitter, Linkedin, Github...)
      - Magic links
    - SSO
    - MFA
    - Security measures
      - Account lockout
      - New device recognition
      - Captchas
  - Authorization
    - Roles
    - Permissions
    - Groups
- Subscription
  - Billing
  - Plans
  - Invoicing
- Analytics
  - KPIS
  - Dashboards
  - Audit logs
- Notifications
  - Email
  - SMS
  - Webhooks
